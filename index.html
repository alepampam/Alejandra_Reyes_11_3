☆ ¿Qué es HTML? ୭ ˚. ᵎᵎ

HTML, o HyperText Markup Language, es el lenguaje con el que se crean las páginas web. Es un lenguaje de marcado, no de programación, que define la estructura y el contenido de un sitio.

☆ Estructura Básica de un Documento HTML ୭ ˚. ᵎᵎ

<!DOCTYPE html>
<html lang="es">
  <head>
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

Esta es la estructura mínima que debe tener todo documento HTML válido.

𓂃˖˳·˖ ִֶָ ⋆Etiquetas Comunes⋆ ִֶָ˖·˳˖𓂃 ִֶָ

  • <h1>...</h1> a <h6>...</h6>: Encabezados jerárquicos
  • <p>: Párrafos de texto
  • <a href="">: Enlaces
  • <img src="">: Imágenes
  • <ul>, <ol>, <li>: Listas

⋆⭒˚｡⋆Buenas Prácticas⋆⭒˚｡⋆

  1. Utiliza una indentación clara y coherente.
  2. Agrega comentarios para organizar el código.
  3. Usa etiquetas semánticas como <header>, <main>, <footer>.
  4. Mantén los nombres de archivo en minúsculas y sin espacios.

⋆⭒˚.⋆¿Qué es script.js?⋆⭒˚.⋆

Es un archivo JavaScript que se utiliza para agregar interactividad a tu sitio web. Puedes usarlo para mostrar mensajes, validar formularios, cambiar estilos dinámicamente, entre muchas otras funciones.

// Ejemplo básico
function saludar() {
  alert("¡Hola, bienvenido a HTML!");
}

⋆⭒˚.⋆¿Qué es PHP?⋆⭒˚.⋆

PHP es un lenguaje de programación del lado del servidor. Se utiliza para crear páginas web dinámicas, conectarse a bases de datos y generar contenido personalizado.

<?php
  echo "¡Hola desde PHP!";
?>

⋆｡°· ¿Qué es style.css?⋆｡°·

Es una hoja de estilos en cascada que define cómo se ve tu página HTML: colores, márgenes, fuentes, posiciones, animaciones y más.

body {
  background: #fff5f9;
  font-family: 'Segoe UI', sans-serif;
  color: #333;
}
